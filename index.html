Header

indexOf

Mit indexOf kann man innerhalb eines Strings die Position eines definierten Strings gesucht. Dessen Position wird dann ausgegeben.

substr

Mit substr kann man einen Teilstring aus einem größeren String extraieren.
let name = 'Andreas Vondenhoff'; let nachname = name.substr(8); let vorname = name.substr(0,7); console.log(nachname); console.log(vorname);
name.substr(0, 7) ==> 0=Start und 7 = Länge
in Zukunft name.substring(0, 7) ==> 0 = Start und 7 = Endposition

Mann kann mehrere Anweisungen in einem String verknüpfen. Z.B
nachname = nachname.charAt(0).toUpperCase() + nachname.substr(1).toLowerCase();

charAt

Verwende stattdessen die Funktion charAt. Sie erwartet als Parameter die Position des zu extrahierenden Zeichens. Zum Beispiel:
nachname = nachname.charAt(0).toUpperCase() + nachname.substr(1).toLowerCase();
In diesem Fall wird der erste Buchstabe des Strings "nachname" gesucht, charAt(0). Das toUpperCase dient dazu die Ausgabe im des Wertes in Großbuchstaben auszugeben.

replace

Replaceermöglicht es, innerhalb eines Strings Ersetzungen vorzunehmen. Als ersten Parameter erwartet die Funktion den gesuchten String und als zweiten den Austausch-String.
Codebeispiel:
let produkt = 'Konterkaffee';
let produktBeschreibung = 'Der Konterkaffee ist ideal zum Rausch bekämpfen, Kaufe Konterkaffee';
let neuesProdukt = 'Kaffeebrause'
console.log(produktBeschreibung.replace(produkt, neuesProdukt)); In der ersten Zeile wird der Produktname festgelegt. In der Zweiten Zeile eine Produkbeschreibung. In beiden Zeilen solle "Konterkaffee" gegen den neuen Produktnamen "Kaffeebrause", der in Zeile 4 definiert wird ausgetauscht werden. Mit dem anschließende console.log wird allerdings nur die erste Erwähnung ausgetauscht. Um alle auszutauschen, muss der Code etwas umgeschrieben werden:
let produkt = /Konterkaffee/g; let produktBeschreibung = 'Der Konterkaffee ist ideal zum Rausch bekämpfen, Kaufe Konterkaffee'; let neuesProdukt = 'Kaffeebrause' console.log(produktBeschreibung.replace(produkt, neuesProdukt));
Mit dem der Regular Expression, eingeleitet durch// und dem Buchstaben g in der ersten Zeile (g = global) werden alle in Zeile 1 definierten Strings gegen den neunen getauscht.
Wenn man im zu ersetzenden String ' 'macht kann man falsche Angaben entfernen.

include

Die erhält als ersten Parameter den Search String, nach welchem gesucht werden soll. Optional kannst du auch noch einen zweiten Parameter in Form eines Startindexes für die Suche angeben. Wenn du keinen Startindex übergibst, wird 0 impliziert.
Mit includes() sieht der Code dann so aus:
let produkt = 'Konterkaffee'; let search = "Konter" console.log(produkt.toLowerCase().includes(search.toLowerCase())); Hier wurde noch über toLowerCase noch die Groß und Kleinschreibung berücksichtigt. Das ist Grundlage einer Suchfunktion.

startWith

Der Aufbau der Funktion startsWith() ähnelt ebenfalls dem von includes(). Der erste benötigte Parameter ist wieder der gesuchte Teilstring.
Ebenfalls kannst du optional auch noch einen zweiten Parameter in Form eines Startindexes für die Suche angeben.
Wenn du keinen Startindex übergibst, wird 0 impliziert. Der Rückgabewert ist true, wenn der gesuchte String zu Beginn des durchsuchten Strings gefunden wird. Ist dem nicht der Fall, wird false zurückgegeben. startsWith() ist ebenfalls case-sensitive. Du musst also wieder auf Groß- und Kleinschreibung achten.
